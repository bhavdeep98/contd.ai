# Contd Community

Welcome to the Contd community! We're excited to have you here.

## Getting Help

### Discord
Join our Discord server for real-time discussions, help, and community events.
- **Link**: [discord.gg/contd](https://discord.gg/contd)
- **Channels**:
  - `#general` - General discussions
  - `#help` - Get help with Contd
  - `#showcase` - Share what you've built
  - `#announcements` - Official updates

### GitHub Discussions
For longer-form discussions and Q&A:
- [GitHub Discussions](https://github.com/contd/contd/discussions)

### Stack Overflow
Ask questions with the `contd` tag:
- [Stack Overflow - contd](https://stackoverflow.com/questions/tagged/contd)
